# C-Projects
Project Title:- "File Hashing & Integrity checker"

Features:-

1. The system generates a unique hash for any selected file.

2. It allows users to verify whether a file has been modified.

3. The tool supports multiple hashing algorithms like MD5 and SHA-256.

4. Users can compare old and new hash values easily.

5. It provides a simple and clean command-line interface.

6. The system can process multiple files one after another.

7. Users can save generated hashes for future verification.

8. It displays the verification result instantly in the terminal.

9. The tool highlights whether the file is safe or changed.

10. It shows the time taken to generate the hash.

11. The system allows exporting hash results to a text file.

12. It supports drag-and-drop file paths into the terminal.

13. The tool alerts users in case of invalid or missing files.

14. It maintains a history of previous hash checks.

15. The system ensures fast and lightweight hashing performance.

16. It works on all major operating systems.

17. The tool offers secure hashing without exposing file data.

18. It provides optional detailed output for advanced users.

19. The system can run in offline mode without internet.

20.  It supports command-line flags for quick operations.
